Multi-Tenant Isolation

Bound tenants within absolute chroot-like path sandboxes under /sandbox/<username>. Restrict file manager access, execution commands, and enforce storage quotas per user.

SQL Client & Explorer

Skip phpMyAdmin. Create databases, define independent DB users, and execute custom SQL queries directly inside an integrated, live table inspector and query editor.

WebServer Orchestration

Switch between NGINX and Apache virtual host engines. Update PHP versions (8.1, 8.2, 8.3) on the fly, toggle sites online/offline, and manage records in an integrated DNS zone file explorer.

FTP & SSH Keys

Provision custom sandboxed FTP accounts. Upload SSH public keys directly to authorization files to facilitate secure, passwordless terminal administration.

Cron Task Scheduling

Configure daemon schedules with cron expressions. Schedule backup cycles, script execution triggers, and automated system maintenance routines.

SSL Auto-Renewals

Provision zero-config Let's Encrypt certificates automatically. Keep track of certificate issuer details and trigger manuals renewals with a single click.

Directory Privacy Locks

Password protect sensitive web directories instantly. Restrict client folder access by assigning usernames and encrypted passwords in a simple visual checklist.

Git Repository Deployer

Deploy code directly into sandboxed directories. Enter a repository URL and branch name to automatically clone source code and trigger live web layouts.

Secure Directory Bounding (Chroot Sandboxing)

Keel Panel keeps your server secure by automatically enforcing directory path containment. Tenants can never escape their sandboxed folder structure, preventing cross-profile security breaches.

  • Absolute Bounding Any file read/write API checks path boundaries recursively to prevent directory traversal attacks (..).
  • Independent Process Context Virtual hosts run under PHP-FPM pools matching the tenant's user profile to isolate execution permissions.
  • Quota Enforcement Server-level folder quotas block writing actions once the tenant reaches their pre-configured limits.
